# Simple Shell

This project is a simple UNIX command line interpreter written in C.

## Description
The simple shell reads commands from standard input and executes them using
the execve system call. It supports both interactive and non-interactive modes.

## Features
- Executes commands with absolute paths
- Handles errors like /bin/sh
- Works in interactive and non-interactive mode
